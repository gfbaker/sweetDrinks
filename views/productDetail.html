Apto para guarda de unos 5 años. Decantar por espacio de una hora. La temperatura adecuada para disfrutar este vino es entre 16º y 18º C. Este vino acompaña muy bien platos a base de carnes rojas, carnes de caza mayor y quesos duros con carácter. En boca se perciben taninos equilibrados. Aterciopelado. De buena estructura, fino, moderno.
